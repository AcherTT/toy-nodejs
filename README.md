# toy-nodejs
一个玩具版本的nodejs，属于个人练习项目，仅支持持linux

## 依赖

1. v8，版本与 node18.15.0 一致使用 10.2 版本
2. libev，使用 4.0 版本


## 编译方式
```shell
mkdir build && cd build
cmake ..
make
```

## 运行方式
```shell
./Node
```
