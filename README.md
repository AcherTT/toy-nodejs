# toy-nodejs
一个玩具版本的nodejs，属于个人练习项目，仅支持持linux

## 依赖

1. v8
2. libev


## 编译方式
```shell
mkdir build && cd build
cmake ..
make
```

## 运行方式
```shell
./Node
```
